**Code**    
A version of the GEOS-Chem adjoint model code used for methane oxidation chain four-dimensional variational (4D-Var) inversions. Assimilated tracers include methane, HCHO, CO, and MCF. Zenodo, https://doi.org/10.5281/zenodo.19823597, 2026. [Available Here](https://zenodo.org/records/19823597)

**Description and citation**   
Yu, X. (2026). Methane Oxidation Chain Data Assimilation in GEOS-Chem Adjoint. Zenodo. https://doi.org/10.5281/zenodo.19823597<br>
Yu, X., Millet, D. B., Henze, D. K., Worden, H., Vigouroux, C., Patra, P., Hill, J. D., & Jackson, R. B. (2026). Multi-species constraints resolve methane source versus sink changes: lessons from 2020. Earth's Future, in press.
